### Create Vue
* required: ts-node
* instaration
    1. GLOBAL INSTALL
        1. Install ts-node
            ```sh
            yarn global add typescript ts-node
            ```
        2. Add PATH `yarn global bin`
    2. or LOCAL INSTALL
        1. Install ts-node
            ```sh
            yarn add typescript ts-node
            ```

```
yarn create:view --name path/to/view
```

```
yarn create:compo --name path/to/component
```
